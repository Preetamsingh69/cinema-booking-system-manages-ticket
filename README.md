# 🎬 Cinema Ticket Booking System

## 📜 Description
A Java-based Cinema Ticket Booking System for efficient reservation management. It maintains a strictly sorted array upon insertion and implements Binary Search for rapid O(log n) retrieval of customer data. Built to demonstrate optimal data structure manipulation.

## 🚀 Features
* Adds new ticket bookings dynamically.
* Automatically sorts records during insertion.
* Searches records efficiently using Binary Search.
* Displays all active reservations in sorted order.

## 🛠️ Tech Stack
* **Language:** Java
* **Data Structure:** Array
* **Algorithms:** Binary Search, Insertion Logic

## ⏱️ Complexity
* **Insertion Time:** O(n)
* **Search Time:** O(log n)
* **Space Complexity:** O(n)

## 💻 How to Run
* Clone this repository.
* Compile the Java file: `javac Main.java`
* Run the program: `java Main`
* Follow the interactive terminal menu.
